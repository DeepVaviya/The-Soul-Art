# 🌿 The Soul Art - Artist Portfolio Website

![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white)

A professional, multi-page portfolio website designed to showcase creative artistry and design services. The site is beautifully categorized to display various collections such as Bridal, Engagement, Festive, Indian, Designer, and Baby Shower designs. 

Built with clean, vanilla web technologies, this site emphasizes fast load times, responsive design, and solid Search Engine Optimization (SEO).

## ✨ Features

- **Categorized Image Gallery:** A dedicated visual portfolio featuring high-quality images organized by event type (Bridal, Engagement, Festive, etc.).
- **Multi-Page Architecture:** Clean navigation separating the Home, About, Services, Gallery, and Contact sections for a professional user experience.
- **Dynamic Testimonials:** Client reviews and feedback managed dynamically via custom JavaScript.
- **SEO & Performance Ready:** - Includes `sitemap.xml` and `robots.txt` for search engine crawling.
  - Pre-configured Open Graph tags for appealing social media sharing.
  - Custom optimized image loading for faster performance.
- **Fully Responsive:** Adapts flawlessly to desktop, tablet, and mobile viewing environments.

## 📁 Project Structure

```text
the-soul-art/
├── images/                 # Extensive categorized image gallery
│   ├── baby-shower/
│   ├── bridal/
│   ├── designer/
│   ├── engagement/
│   ├── festive/
│   └── indian/
├── index.html              # Main landing page
├── about.html              # Artist biography and journey
├── services.html           # Details of offered packages and services
├── gallery.html            # Visual portfolio of works
├── contact.html            # Booking and contact information
├── styles.css              # Global styling and responsive layouts
├── main.js                 # General site interactivity and DOM manipulation
├── testimonials.js         # Logic for rendering client testimonials
├── compress-images.js      # Utility script for bulk image compression
├── robots.txt              # Search engine crawler instructions
└── sitemap.xml             # XML sitemap for SEO indexing
```
